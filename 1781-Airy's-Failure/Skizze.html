$ v $: Angebliche Geschwindigkeit der Erde.
$ w $: Strecke, die die Erde in der Zeit angeblich zurücklegt, in welcher ein »Lichtpartikel« durch das Teleskop geht.
$ c $: Lichtgeschwindigkeit.
$ ℓ $: Länge des Teleskopes.
$ s $: Tatsächliche Strecke, die ein »Lichtpartikel« im Teleskop verbringt.

Ein paar Zusammenhänge

Mit Cosinussatz: $$ s^2 = w^2 + ℓ^2 - 2wℓ\cos(\tau) $$ $$ ℓ^2 = w^2 + s^2 = 2wℓ\cos(\sigma) $$ Durch Umformung (Beide Gleichungen enthalten $w^2$): $$ cos(\tau) = { w + s\cos(\sigma) \over ℓ } $$ Das Licht benötigt für die Strecke $s$ gleich lang wie die Erde für die Strecke $w$: $$ w = {c s \over v } $$

Licht direkt von oben

Unter der Annahme, dass $ \sigma={\pi \over 2} $, also, dass das Licht gerade »von oben« scheint, gilt (da $ cos({\pi \over 2}) \equiv 0 $) $$ cos(\tau) = { w \over ℓ } $$ $$ w = ℓ\cos(\tau) $$ $$ w\tan(\tau) = s $$ $$ tan(\tau) = {s \over w} = { sv \over cs } = { v \over c } $$
